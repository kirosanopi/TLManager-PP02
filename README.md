:: EN_MAN ::
The TLManager project is a full-fledged application with all CRUD-endpoints, which is an interactive list of notes with an intuitive and pleasant interface.

Instructions for using the project:

Install project on your computer. Run the servers-open.bat file. Two console shells will open, each of which is responsible for its own server, i.e. backend and frontend. First, the backend-server window will open, followed by the frontend-server window. In the second window, you will need to press Y to continue. The server is now ready to use.

In case the script doesn't work, we start the servers manually: 

1. Install the project in the selected directory on your computer;
2. Open any console;
3. In the console, go to the local backend folder and run the command: "npm start";
4. Open a new console shell;
5. Go to the frontend directory and run the same command: "npm start", then click Y when the system asks, and we are automatically redirected to the site.

:: RU_MAN ::
Проект TLManager - это полноценное приложение со всеми CRUD-endpoints, представляющее собой интерактивный список заметок с интуитивно понятным и приятным интерфейсом.

Инструкция по эксплуатации проекта:

Устанавите проект на свой компьютер. Запустите файл servers-open.bat. Откроется две консольные оболочки, каждая из которых отвечает за свой сервер, то есть backend and frontend. Вначале откроется окно с запуском backend-server, затем уже frontend-server. Во втором окне необходимо будет нажать Y, чтобы продолжить. Все, сервер готов к работе;

В случае, если скрипт не сработает, запускаем сервера вручную: 

1. Устанавливаем проект в выбранный каталог на вашем компьютере;
2. Открываем любую консоль;
3. В консоли переходим в локальную папку backend и выполняем команду npm start;
4. Открываем новую консольную оболчку;
5. Переходим в каталог frontend и выполняем ту же команду npm start, затем жмем Y, когда этого попросит система и нас автоматически перессылает на сайт.
